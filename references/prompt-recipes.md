# Prompt Recipes

## Screen Prompt

Design an EVA-inspired monitoring interface with red-orange command colors, hard-edged frames, condensed technical typography, high information density, and non-typographic pattern language built from stripe bands, ticks, calibration rails, threshold bars, and reticle-like brackets.

## Component Prompt

Create a reusable instrumentation component for an EVA-style control system. Use hard borders, compact labels, tabular data, explicit normal / warning / critical states, and at least one structural motif such as stripes, ticks, threshold segments, or targeting brackets.

## Review Prompt

Review this interface for EVA fidelity. Flag any place where the design drifts into generic sci-fi or modern SaaS dashboard patterns. Pay special attention to color semantics, data density, border language, measurement marks, and pattern primitives.
